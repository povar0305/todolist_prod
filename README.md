# to_do_list
Проект тодо листа. Включает в себя создание записи, отчетка записи как сделанной, поиск по записям и сортировка по дате или статусу.

Использовался vue2, bootstrap vue, vuetifyjs
## Установка проекта
```
npm install
```

### Запуск разработки
```
npm run serve
```

### Запуск билда
```
npm run build
```
